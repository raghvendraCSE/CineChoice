# 🎬 Movie Recommendation Web App

A fully responsive Movie Search and Watchlist Web Application built using **HTML, CSS, and JavaScript** with real-time integration of the **OMDb API**.

This application allows users to search for movies, view detailed movie information in a modal popup, and save their favorite movies using a persistent Watchlist feature powered by localStorage.

## 📌 Features

- 🔎 Search movies using OMDb API
- 🎬 View detailed movie information in a popup modal
- ⭐ Display IMDb rating, genre, director, cast, and plot
- ❤️ Add movies to Watchlist (saved in browser)
- ⚡ Client-side caching for improved performance
- 📱 Fully responsive design (Mobile, Tablet, Desktop)
- 🎨 Interactive hover effects and smooth UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Responsive Design, CSS Grid)
- JavaScript (ES6+)
- OMDb API
- LocalStorage

---

## 🧠 How It Works

1. User enters a movie name in the search bar.
2. The app fetches movie data from the OMDb API.
3. Search results are dynamically displayed.
4. Clicking a movie opens a detailed modal.
5. Users can add movies to their Watchlist.
6. Watchlist data is stored in the browser using localStorage.

## ⚙️ Setup Instructions

1. Clone the repository:

```

```

2. Navigate to the project folder:

```
cd movie-app
```

3. Replace the API key inside `script.js`:

```javascript

```

4. Open `index.html` using Live Server or any browser.
